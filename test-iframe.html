üêõ Troubleshooting

Issue: iframe doesn't load

  • Check browser console for error messages
  • Look for "Refused to display in a frame" or CSP violation errors
  • Verify X-Frame-Options is NOT present in response headers
  • Verify Content-Security-Policy frame-ancestors includes your domain

Issue: API calls fail

  • Check for CORS errors in browser console
  • Verify Access-Control-Allow-Origin header is present
  • Ensure the origin matches exactly (protocol and domain)
